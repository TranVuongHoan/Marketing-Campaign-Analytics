# 📢 Digital Marketing Campaign Analytics

This repository contains a comprehensive analysis of 500 digital marketing campaigns extracted from `06_Marketing_Digital_Campaign.xlsx`. Total spend analyzed: **100 Billion VND** across 10 platforms, 10 brands, and 6 cities in Vietnam.

---

## 📁 Repository Structure

| Folder | Contents |
|--------|----------|
| `1. Data Preparation/` | Dataset overview, field descriptions |
| `2. Data Cleaning/` | Data quality issues and normalization |
| `3. Data Analysis/` | 20 SQL queries + 7 Python EDA charts |
| `4. Report Creation/` | Interactive HTML dashboard |

---

## 🔑 Key Findings

- **Email ROAS: 4,572×** vs Programmatic ROAS: 6.3× — a **727× performance gap** with equal budget
- **Email + SMS generate 97% of total revenue** (56.74T VND) despite only 20% of budget
- **Month 8 golden window:** ROAS 1,292× — but receives less budget than Month 7 (ROAS 190×)
- **HCMC efficiency (798×)** receives same budget as Can Tho (412×) — geo allocation misaligned
- **Publicis Vietnam ROAS (1,072×)** is 3× higher than TBWA Vietnam (351×)
